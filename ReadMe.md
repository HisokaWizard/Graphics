ЦельСоздать на основе предоставленных данных прототип для визуализации результатов реакции ПЦРФункциональные требования1.	Создать 3 представления для данных см. Скриншот):a.	Представление в виде графика – обязательное. График интерактивный. Каждая кривая может быть подсвечена при наведении и выбрана при щелчкеb.	Представление образцов на плашке – обязательное (в каждой ячейке в реакции ПЦР устанавливается капсула с образцом) c.	Табличное представление – по возможности. В этом представлении отражено расположение образцов на (координаты ячейки на плашке) плашке, его CQ2.	Все три представления должны быть синхронизованы друг с другом, то есть при наведении на кривую на графике, подсвечивается соответствующая ячейка на плашке и подсвечивается соответствующая строка табличного вида.  3.	Входные данные представлены в переданном нами файле Microsoft Excel – «140221-vve-TV-NG-CT-VKO multiplex -  Quantification Amplification Results.xlsx»4.	Желательно, чтобы прототип имел возможность самостоятельно считать CQ. CQ- этот тот цикл амплификации, при котором происходит максимальный прирост количества копий образца взятый за все время протекания реакции. 5.	Для иллюстрации роста количества копий с каждым циклом прикладываем отдельный файл «Расчет CQ.XLSX», который благодаря графику  должен навести на мысль о том, по какой формуле его считать Скриншот 1. Представление в виде графика – левый верхний угол, представление в виде плашки – левый нижний угол, представление в виде таблицы – правый нижний угол.Технические требования1.	Необходимое, чтобы прототип работал в веб-браузере.2.	Допускается использование любого наиболее удобного или наиболее знакомого фреймворка (angular, react, etc.)3.	Язык программирования - JavaScriptОжидаемое время выполнения6-8 часовЕсли фактическое время превышает 8 часов, то необходимо связаться с нами и показать промежуточный результат.
